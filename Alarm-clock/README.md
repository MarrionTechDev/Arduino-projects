### Arduino Alarm Clock
A simple alarm clock built using Arduino and basic hardware components.

##Components
- Arduino UNO
- Breadboard
- LED display MAX7219 8x32 Dot Matrix
- Male and Female Jumper Wires
- RTC Module Clock DS3231
- Power Cable for Arduino

## Software
- Arduino IDE

## Features
- Display the time in 12-hour format
- Displays the am and pm accordingly

## Future Improvements
- Set and trigger an alarm at a specified time
- Manual stop/reset with buttons
- Timed logic for buzzer activation
- Add a snooze function
- Build a fully portable design
